The ubuntu user has no rights on the HDFS file system
to initialize a home folder for it use the following commands:

sudo -u hdfs -i hadoop fs -mkdir /user/ubuntu
sudo -u hdfs -i hadoop fs -chown ubuntu /user/ubuntu
sudo -u hdfs -i hadoop fs -chmod 750 /user/ubuntu

Optionally set a quota
sudo -u hdfs -i hdfs dfsadmin -setSpaceQuota 5497559962838 /user/ubuntu


To list and Kill Hadoop jobs from the shell use this commands:
mapred job -list
mapred job -kill job_1470985755205_0007


To hack the core-site.xml to add S3 credentials use this command:

sudo -u hdfs vim /etc/hadoop/2.4.2.0-258/0/core-site.xml
